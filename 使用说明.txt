驱动库与镜像
------------
  仓库: https://github.com/yanbai9188-oss/qudong
  manifest: v1.3.0（release: v1.1.0 统一 URL）
  Release: v1.1.0（26 个驱动 ZIP，含扩展包）

  填充本地库: 填充驱动库.bat
  Win10 常见包: 填充Win10驱动.bat
  扩展包(NVIDIA/触摸板/打印机): 填充扩展驱动.bat
  发布驱动仓库: 发布驱动仓库.bat
  或: scripts\Publish-DriverRepo.ps1 -DryRun
  或: scripts\Publish-DriverRepo.ps1 -AddNewPackages -ManifestVersion 1.4.0 -ReleaseTag v1.2.0
  或: scripts\Publish-DriverRepo.ps1 -UploadRelease -PushManifest -ReleaseTag v1.1.0
  发布 Release v1.1.0: 发布Release_v1.1.0.bat
  或: qudong-repo\Publish-Release.ps1 -Tag v1.1.0

支持系统
--------
  主要: Windows 10 x64（Build 10240+）
  兼容: Windows 11 x64
  需要: PowerShell 5.1+、管理员权限（安装时）

启动
----
  推荐: 安装 release\CIODIY_DriverBooster_Setup_1.2.3.exe（双击安装，桌面快捷方式启动）
  开发: 双击「启动驱动检测安装.bat」
  打包: 双击「打包安装程序.bat」

  装完即用（类似 Driver Booster）:
    - 首次启动自动同步驱动索引（需联网）
    - 扫描 → 一键修复，驱动 ZIP 自动从 GitHub 下载，无需手动拷 Drivers\

界面
----
  场景快捷修复：无声音 / 无网络 / USB（扫描 + 可选立即修复）
  立即扫描 / 一键修复 / 同步镜像 / 安装本地库
  系统栏显示：机型 | Windows 版本 Build | 架构

命令行
------
  .\DriverBooster.ps1 -ScanOnly -NoGui
  .\DriverBooster.ps1 -ScanOnly -NoGui -Scenario audio
  .\DriverBooster.ps1 -FixAll -NoGui -Scenario network -RollbackOnError
  .\DriverBooster.ps1 -HealthOnly -NoGui
  .\DriverBooster.ps1 -DeployMode -NoGui -AutoFix -RebootIfNeeded
  .\DriverBooster.ps1 -DeployMode -NoGui -DeploySilent
  .\DriverBooster.ps1 -RollbackLast

  退出码（-FixAll -NoGui）: 0=成功  1=失败/回滚  2=成功但建议重启

Win10 批量重装建议流程
----------------------
  1. 同步驱动镜像（拉 manifest v1.1.0）
  2. 扫描缺失驱动
  3. 优先在线下载（GitHub Release）或预置 Drivers\ 离线库
  4. 一键修复（事务 + 验证 + 可回滚）
  5. 重启后再扫描确认
